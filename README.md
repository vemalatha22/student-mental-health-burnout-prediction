 ## Student Mental Health Burnout Prediction System

## Overview

This project focuses on predicting student mental health burnout levels using Machine Learning techniques. The aim of the project is to analyze how different academic, behavioral, and lifestyle-related factors contribute to student burnout and to build a predictive model based on these patterns.
The project was developed as part of my learning journey in Machine Learning and helped me understand the complete ML workflow, including data preprocessing, exploratory data analysis, model training, evaluation, and interpretation.

 ## Problem Statement

Student burnout has become increasingly common due to academic pressure, stress, unhealthy routines, and mental health challenges. Early identification of burnout patterns can help students and institutions take preventive measures and improve student well-being.
This project uses machine learning to analyze student-related factors and predict burnout scores.

## Objectives

- Analyze factors affecting student burnout
- Perform Exploratory Data Analysis (EDA)
- Preprocess and clean the dataset
- Train a Machine Learning regression model
- Evaluate model performance using regression metrics
- Understand feature influence on burnout prediction

## Model Used

### Linear Regression

Linear Regression was used because the target variable (burnout score) is continuous in nature. The model helps identify relationships between student-related features and burnout levels.

## Evaluation Metrics

The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
